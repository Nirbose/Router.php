# Router.php
 code source de mon routeur php
 
## Utilisation :
Les fonctions disponibles sont :
 - get
 - post
 - put
 - patch
 - delete
 - options

Exemple :
```php
Route::get('about', function () {
  return "<h1>About Page !</h1>";
});
```
